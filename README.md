# Vite React app starter project.

Empty React starter project.

- Bundled with Vite. Yarn as a package manager.
- Uses CSS modules.
- TypeScript.
- ESlint for linting and formatting.

Tested/Created with Node v20.3.0

## How to run

`cd vite-react-starter`

`yarn`

`yarn run dev`
